Hello folks. This is a basic ecommerce API for educational purposes. Feel free to clone and upgrade the project.

### Features


* ✅ JWT auth.
* ✅ Pagination support.
* ✅ Swagger docs.
* ✅ Written in NET.5 and EF Core 5.

### Installation

```
$ dotnet run -p Commerce.API
```
